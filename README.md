# 📊 Madhav Ecommerce Sales Dashboard

An interactive **Power BI dashboard** built to analyze ecommerce sales performance across multiple business dimensions including revenue, profit, customers, products, states, payment methods, and time.

## 🚀 Overview

This dashboard provides a comprehensive view of ecommerce sales data, helping stakeholders monitor key business metrics and identify trends for better decision-making.

## ✨ Features

- 📈 Sales and profit analysis
- 💰 Revenue and profitability KPIs
- 🗺️ State-wise sales performance
- 👥 Customer-wise analysis
- 📦 Product category insights
- 💳 Payment mode distribution
- 📅 Monthly profit trends
- 🎯 Interactive slicers for filtering data

## 📊 Dashboard Insights

The dashboard includes:

- **KPI Cards**
  - Total Sales Amount
  - Total Profit
  - Total Quantity Sold
  - Average Order Value (AOV)

- **Visualizations**
  - Monthly Profit Analysis
  - Sales by State
  - Sales by Customer
  - Quantity by Product Category
  - Quantity by Payment Mode
  - Profit by Sub-Category

- **Interactive Filters**
  - Quarter
  - State

## 🛠️ Built With

- Microsoft Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
